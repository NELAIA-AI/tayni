# NELAIA Core

**NELAIA** — **N**otation for **E**xecution: **L**ayer for **A**gent-**I**nitiated **A**utomation

> An Execution Notation: a format designed for AI agents to emit computational intent directly, without ambiguity, optimized for generation and efficient compilation.
>
> **NELAIA is not a traditional programming language. It is an executable notation for AI.**

## Overview
NELAIA Core implements the foundational components of the NELAIA architecture.

## Features
- AST
- Parser
- Execution engine
- Safety layer
